The ui of the emulex download
===


### Dependences
*  nodejs
*  bower
*  electron

### Install
*  bower 
```
    npm install bower -g
    bower install
```
*  electron
```
    npm install -g electron
```

### Config 
```
    /app/components/config/config.constant.js
```

### Run
```
    electron app/
```
